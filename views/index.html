Timestamp Microservice

A freeCodeCamp project from the Back End Development and APIs Certification

Usage

A request to /api/:date? with a valid date string will return a JSON object containing an Unix timestamp and an UTC string representation of the requested date:

/api/2021-12-29
/api/12-29-2021
/api/1640736000000

Output

{"unix": 1640736000000, "utc": "Wed, 29 Dec 2021 00:00:00 GMT"}

Note: An empty string request will return information for the current date. Invalid date strings will return an error object:

/api
/api/not-a-date-string
